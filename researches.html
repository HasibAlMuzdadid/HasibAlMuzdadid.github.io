Title Comparative Analysis of Different Feature Extraction Techniques for Hyperspectral Image Classification.
Authors Md Hasib Al Muzdadid Haque Himel, Sadia Zaman Mishu (Supervisor)

Abstract Hyperspectral imaging has been an important field in many application areas such as ground cover classification, object detection, mining, geology and so on. Hyperspectral sensors collect hundreds of spectral bands to acquire an image and many of these bands contain irrelevant information. It is a challenging task to process the hyperspectral images because of high dimensionality and data redundancy. Classification technique can be applied to classify high dimensional data. Before applying classification technique, the data needs to be pre-processed to improve classification accuracy since high dimensional data suffers from Hughes Phenomenon. Dimension can be reduced by using feature extraction techniques. Here, Principal Component Analysis (PCA) a well-known feature extraction technique and Segmented Principal Component Analysis (SPCA) have been used for feature extraction. Then, classification is done using Support Vector Machine (SVM) classifier. For Indian Pines data classification accuracy has been achieved 82.72% and 84.94% and for Pavia University data classification accuracy is 90.27% and 92.95% when applied PCA and SPCA respectively. This thesis report highlights the comparative analysis of these feature extraction techniques for hyperspectral image classification.
